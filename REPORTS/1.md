# Sprint 1 Report: Authentication System Setup

## Overview

Sprint 1 focused on setting up a robust, secure authentication system for the EvalFlow project. By utilizing Laravel Breeze, we successfully scaffolded the necessary views, controllers, and routes to handle user authentication, providing a solid foundation for user-specific features in future sprints.

## Tech Stack Used

- **Backend Framework:** Laravel 11.x
- **Authentication Scaffolding:** Laravel Breeze (Blade Stack)
- **Frontend Stack:** 
  - **Templating:** Laravel Blade
  - **Styling:** Tailwind CSS
  - **JavaScript:** Alpine.js
  - **Bundler:** Vite
- **Database:** MySQL (via XAMPP)

## Features Implemented in Sprint 1

The following features were introduced out of the box with the Breeze installation:

1. **User Registration:** Users can sign up for an account.
2. **User Login & Logout:** Secure session-based authentication.
3. **Password Reset:** Users can request a password reset link (requires mail server configuration in `.env` to fully test).
4. **Email Verification:** Infrastructure for verifying user emails is ready.
5. **User Dashboard:** A protected route (`/dashboard`) that is only accessible to authenticated users.
6. **Profile Management:** Users can update their name, email, and password, as well as delete their account.

## Provided APIs / Routes

The Breeze installation registered several new routes for authentication. You can see all of them by running `php artisan route:list`. The primary endpoints are:

- `GET /register` - Registration view
- `POST /register` - Handle registration submission
- `GET /login` - Login view
- `POST /login` - Handle login submission
- `POST /logout` - Handle logout
- `GET /dashboard` - Protected dashboard view
- `GET /profile` - Profile management view

## How to Test

To test the features implemented in this sprint:

1. **Start the Laravel server:**
   ```bash
   php artisan serve
   ```
2. **Start the Vite development server (in a separate terminal):**
   ```bash
   npm run dev
   ```
3. **Navigate to the application:** Open `http://127.0.0.1:8000` in your web browser.
4. **Test Registration:** Click on the "Register" link in the top right corner. Fill out the form to create a new user account. You should be redirected to the Dashboard.
5. **Test Logout/Login:** Click on your name in the top right corner of the dashboard, select "Log Out", and then try logging back in using the "Log In" link.
6. **Test Profile:** Navigate to `http://127.0.0.1:8000/profile` to update your user information.
